# CH-1. ENCODING SCHEMES AND NUMBER SYSTEM

## 1. Encoding
- Encoding is the process of converting data, such as text, numbers, or symbols, into a specific format or code so that a computer can store, process, and transmit it correctly.

<br>

## 2. Number System
- A Number System is a method of representing numbers using a set of digits according to a specific base (radix). Different number systems use different digits. Computers mainly use the Binary Number System because electronic circuits work with only two states: ON (1) and OFF (0).

<br><br>

## ASCII (AMERICAN STANDARD CODE FOR INFORMATION INTERCHANGE)
- ASCII is a character encoding standard that assigns a unique numeric code to each letter, digit, punctuation mark, and special symbol so that computers can store, process, and exchange text. ex:- A → 65, a → 97, 0 → 48

## ISCII (Indian Standard Code for Information Interchange)
- ISCII (Indian Standard Code for Information Interchange) is a character encoding standard developed to represent and exchange text in Indian languages such as Hindi, Bengali, Tamil, Telugu, Punjabi, Gujarati, Kannada, Malayalam, Odia, and others. 

## UNICODE 
- Unicode is a universal character encoding standard that assigns a unique code to every character, symbol, and emoji from almost all languages of the world, allowing computers to store, process, and exchange text correctly. Common Unicode encoding formats include UTF-8, UTF-16, and UTF-32.

<br><br>

## NUMBER SYSTEM
- A Number System is a method of representing numbers using a set of digits based on a specific base (radix). Different number systems use different bases and sets of digits.

## Types of Number Systems

### 1. Decimal Number System
- The Decimal Number System is a number system with base (radix) 10. It uses 10 digits (0–9) to represent numbers and is the most commonly used number system in everyday life.

### 2. Binary Number System
- The Binary Number System is a number system with base (radix) 2. It uses only two digits, 0 and 1, to represent numbers and is the number system used by computers.

### 3. Octal Number System
- The Octal Number System is a number system with base (radix) 8. It uses eight digits (0–7) to represent numbers.

### 4. Hexadecimal Number System
- The Hexadecimal Number System is a number system with base (radix) 16. It uses 16 symbols: 0–9 and A–F, where A = 10, B = 11, C = 12, D = 13, E = 14, and F = 15. Mostly used in :- Web color codes , Computer hardware , Memory addresses, etc
