> ## CPU
- The CPU (Central Processing Unit) is the main processing unit and the brain of the computer. It executes instructions, processes data, performs calculations, and controls the working of all other components of the computer.


### its three main parts are :
    
    1. The ALU (Arithmetic Logic Unit) is a part of the CPU that performs arithmetic operations like addition, subtraction, multiplication, and division, and logical operations like comparison and decision-making.

    2. The CU (Control Unit) is a part of the CPU that controls and coordinates all the operations of the computer. It manages the flow of data and instructions between different components of the computer.

    3. Registers are small, high-speed memory locations inside the CPU that temporarily store data, instructions, and intermediate results for fast processing.

### IN EASY WORDS

| | |
|---|---|
| ALU | Performs calculations and logical comparisons |
| CU | Controls and coordinates all operations |
| Registers | Temporarily store data and instructions |

> ## Evolution of Computer

```
The evolution of computers refers to the development of computers through different generations. With each generation, computers became smaller, faster, more reliable, more powerful, and more efficient due to improvements in technology.
```

### Generations of Computers
| Generation | Main Technology | Features |
|------------|-----------------|----------|
| **First Generation (1940–1956)** | Vacuum Tubes | Very large, slow, expensive, produced a lot of heat. |
| **Second Generation (1956–1963)** | Transistors | Smaller, faster, more reliable, generated less heat. |

| **Fourth Generation (1971–Present)** | Microprocessors | Very small, high speed, affordable, introduced personal computers. |
| **Fifth Generation (Present & Future)** | Artificial Intelligence (AI) | Intelligent systems, natural language processing, robotics, machine learning. |

> ## COMPUTER MEMORY
```
Computer memory is the storage unit of a computer used to store data, instructions, and information. It helps the computer save, retrieve, and process data. Computer memory is mainly of two types: Primary Memory (RAM and ROM) and Secondary Memory (Hard Disk, SSD, Pen Drive, etc.).
```

### Types of Memory

#### 1. Primary Memory (Main Memory)
* Primary memory is the main memory of a computer that is directly accessed by the CPU. It stores data and instructions that are currently being used by the computer so that processing can happen quickly.


#### 2. Secondary Memory 
* Secondary memory is used to store data permanently. It is not directly accessed by the CPU and has a larger storage capacity than primary memory.

#### 3. Cache Memory
* Cache memory is a small, very high-speed memory located inside or near the CPU. It stores frequently used data and instructions so that the CPU can access them quickly, thereby improving the speed and performance of the computer.

| Primary Memory | Secondary Memory |
|----------------|------------------|
| Accessed directly by the CPU | Not accessed directly by the CPU |
| Faster | Slower |
| Smaller storage capacity | Larger storage capacity |
| Stores currently used data | Stores data permanently |
| Examples: RAM, ROM | Examples: HDD, SSD, Pen Drive |

## Microprocessor
- A microprocessor is a small electronic chip that contains the CPU (Central Processing Unit) of a computer on a single integrated circuit (IC). It processes data, executes instructions, and controls all the operations of the computer.


### Main Specifications of a Microprocessor
1. Clock Speed
    - Definition: The speed at which a microprocessor executes instructions.
    - Unit: Hertz (Hz), Megahertz (MHz), Gigahertz (GHz).
    - Important Point: A higher clock speed generally means the processor can execute more instructions per second.
    - Example: 3.2 GHz

2. Word Size
    - Definition: The number of bits a processor can process at one time.
    - Common Word Lengths: 32-bit, 64-bit.
    - Important Point: A larger word length allows the processor to handle more data in one operation.

3. Cores
    - Definition: The number of processing units (cores) inside a microprocessor.
    - Examples: Dual-core (2), Quad-core (4), Hexa-core (6), Octa-core (8).
    - Important Point: More cores help the processor perform multiple tasks simultaneously.

4. Memory Size
    - Definition: The maximum amount of memory (RAM) that a microprocessor can access or support.
    - Examples: 32-bit processor, 64-bit processor
    - Important Point: A larger memory size allows the microprocessor to handle more data, run larger programs, and improve the computer's overall performance.

> ### Data Types

#### A. Structured Data
- Structured data is data that is organized in a fixed format, such as rows and columns, making it easy to store, search, and process. ex : Student database , Excel spreadsheet

#### B. Unstructured Data
- Unstructured data is data that does not have a fixed format or predefined structure. It is difficult to organize and process using traditional databases. ex : Images , Videos 

#### B. Semi-Structured Data
- Semi-structured data is data that does not follow a fixed table format but contains tags, labels, or keys that help organize and identify the data. ex : JSON files , HTML documents

> ## SoftWare
- Software is a collection of programs and instructions that tells a computer what to do and how to perform specific tasks. It enables the computer hardware to function properly.

### Types of Software

#### 1. System Software
- System software is software that manages and controls the computer hardware and provides a platform for application software to run.
    * a. **Operating System** : An Operating System (OS) is system software that manages the computer's hardware and software resources. It acts as an interface between the user and the computer, allowing users to run applications and perform various tasks. ex: Microsoft Windows , Linux 
    * b. **Device Drivers** : A device driver is a system software that enables the operating system to communicate with and control hardware devices. It ensures that the connected hardware functions correctly. ex : Printer Driver , Keyboard Driver
    * c. **System utilities** : System utilities are system software programs that help maintain, manage, protect, and optimize the performance of a computer system. They improve the efficiency and reliability of the computer. ex : Antivirus Software , Backup Utility 

#### 2. Application software
- Application software is software designed to help users perform specific tasks. It runs on system software and is used for activities such as writing documents, browsing the internet, or playing media.ex : Instagram , Youtube , etc . It can be Divided Into two broad categories those are as follows :-
    * a. **General Purpose Software** : General Purpose Software is application software designed to perform common tasks for a large number of users. It can be used in different fields and for various purposes. ex : MS Word , MS Excel
    * b. **Customised Software** : Customized Software is application software developed to meet the specific requirements of a particular user or organization. It is designed to perform specialized tasks according to the user's needs. ex : School Management System , Hospital Management System

## Operating System (OS)
```
An Operating System (OS) is system software that acts as an interface between the user and the computer hardware. It manages all the computer's resources, such as the CPU, memory, files, and input/output devices, and provides an environment for running application programs.
```

### OS User Interface
- An OS User Interface is the medium through which a user interacts with the operating system. It allows users to give commands and receive results. It's Types are as follows :-
    1. **Command-Based Interface** :- A Command-Based Interface or Command-Line Interface (CLI) is a type of user interface in which the user types text commands using a keyboard to interact with the operating system.Example: Windows Command Prompt, Linux Terminal
    2. **Graphical-Based Interface** :- A Graphical User Interface (GUI) is a type of user interface that allows users to interact with the computer using graphical elements such as icons, menus, buttons, and windows instead of typing commands.Example: Microsoft Windows, macOS, Android
    3. **Touch-Based Interface** :- A Touch-Based Interface is a user interface that allows users to interact with a device by touching the screen using their fingers or a stylus. Example: Smartphones, Tablets, ATMs
    3. **Touch-Based Interface** :- A Voice-Based Interface is a user interface that allows users to give voice commands to control a computer or device using speech recognition technology. Example: Siri, Google Assistant, Alexa
    3. **Touch-Based Interface** :- A Gesture-Based Interface is a user interface that allows users to interact with a device using hand movements, body gestures, or facial expressions without touching the device. Example: Microsoft Kinect, Gesture controls in smartphones, VR headsets

<br>

### Functions of Operating System (OS)

#### 1. Process Management 
- Process Management is the function of an operating system that creates, schedules, executes, and terminates processes. It also allocates the CPU to different processes so that they run efficiently.

#### 2. Memory Management
- Memory Management is the function of an operating system that allocates, tracks, and deallocates memory for programs and processes. It ensures that memory is used efficiently.

#### 3. File Management
- File Management is the function of an operating system that creates, stores, organizes, retrieves, renames, copies, moves, and deletes files and folders.

#### 4. Device Management
- Device Management is the function of an operating system that controls and coordinates input and output (I/O) devices such as the keyboard, mouse, printer, scanner, and monitor. It uses device drivers to communicate with hardware.
